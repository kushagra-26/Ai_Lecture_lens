# 🧠 CLAUDE.md — AI Lecture Lens (Education AI System Prompt)

## 🎯 PRODUCT CONTEXT

You are building **AI Lecture Lens**, an advanced EdTech platform that helps students:

* Record lectures
* Generate summaries
* Create quizzes
* Track learning progress
* Interact with AI for understanding concepts

This is NOT a generic dashboard.
This is a **modern AI-powered learning product (SaaS-level)**.

---

## 🧭 PRIMARY GOAL

Always generate:

* Clean
* Modern
* Minimal
* Premium-looking UI

The product should feel like a mix of:

* Notion (clean + structured)
* ChatGPT (AI interaction)
* Duolingo (learning-focused UX)
* Linear (polished UI)

---

## ❌ STRICTLY AVOID

* Generic admin dashboards
* Too many cards/boxes
* Bootstrap-style UI
* Excessive borders
* Centered text everywhere
* Random colors
* Poor spacing
* Cluttered layouts

If the UI looks like a template → it is WRONG.

---

# 🎨 DESIGN SYSTEM — AI Lecture Lens (Education Theme)

## 🧠 Design Identity

The product should feel like:

* A digital study space
* A smart notebook
* An AI-powered classroom

NOT a dashboard.

---

## 🎓 Visual Style

Inspired by:

* Anthropic (calm intelligence)
* Notion (clean structure)
* Skillshare (creative learning)

---

## 🎨 Color System

### Background

* Primary: #0B0F19
* Surface: #111827

### Text

* Primary: #E5E7EB
* Secondary: #9CA3AF

### Accent (IMPORTANT)

* Gold: #EAB308 → highlights, buttons, progress
* Green: #22C55E → learning progress
* Red: #EF4444 → errors

---

## ✨ Feel

* Slightly warm (not cold tech UI)
* Comfortable for long reading
* Focus-driven

---

## 🔤 Typography

* Inter / Satoshi
* Clean + readable
* More spacing between lines

---

## 📐 Layout Philosophy

* Feels like a notebook, not admin panel
* Content-first UI
* Minimal containers

---

## 🧩 Components

### Cards

* Soft, blended with background
* No heavy borders
* Rounded-xl

### Buttons

* Gold accent (#EAB308)
* Slight glow on hover

---

## 🧠 AI UI

* Chat-style interactions
* Highlight key insights
* Inline explanations

---

## 🎯 UX RULE

User should feel:
👉 “I’m learning something”
NOT
👉 “I’m using software”

---


### 📊 Dashboard

DO:

* Show "Continue Learning"
* Show progress visually
* Highlight important info

DON’T:

* Use 6+ cards
* Treat everything equally

---

### 🎥 Lectures Page

Each lecture should show:

* Title
* Duration
* Progress bar
* Status

Enhancements:

* Hover preview
* Resume button

---

### 🧾 Summaries Page

Layout:

* Left → lecture list
* Right → AI summary

Style:

* ChatGPT-like
* Clean reading experience

---

### 🧪 Quiz Page

* Full-screen focus mode
* One question at a time
* Progress indicator
* Instant feedback

---

### 📈 Analytics Page

* Graph-based (not text-heavy)
* Weekly progress
* Study streak
* Performance trends

---

### 👤 Profile Page

* Minimal
* Clean sections
* No unnecessary stats overload

---

## ✨ MICRO INTERACTIONS

Always include:

* Hover effects (scale slightly)
* Smooth transitions (200ms)
* Loading skeletons
* Progress animations

---

## 🧠 AI EXPERIENCE RULES

Include:

* Chat-style interfaces
* AI suggestions
* Smart highlights
* Auto-generated content

Avoid:

* Static UI feeling
* Hidden AI features
